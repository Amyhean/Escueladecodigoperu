Tres Módulos

Módulo 0. Pioneras


Te presentamos el Módulo 0. Pioneras. Conociendo el ambiente de computación (M0) Este módulo es de introducción para la participante y sirve para alinear sus conocimientos en Linux o bien, si no lo conoce, tener los conocimientos básicos para comenzar el Módulo Elaboración de sistemas web (MW). Está conformado por tres actividades con un enfoque de sus contenidos por competencia, es por ello que no se especifican objetivos de aprendizaje, sino competencias. Competencia del módulo: Manipular el entorno de trabajo Linux utilizando la Interfaz Gráfica de de Usuario, así como comandos de la consola.

Módulo 1: Construyendo mi primer sitio web.

Comenzaremos por HTML, en el cual aprenderemos a armar la estructura básica de una página web, continuaremos nuestra travesia por el mundo de CSS en donde nuestro esqueleto se viste de colores, aprenderemos los códigos mas importantes para darle la mejor apariencia a nuestra página. Competencia del taller: Diseñar y desarrollar un sitio web estático que pueda ser visto en la web y que se ajuste a los requerimientos levantados con un “cliente”. En este veremos los lenguajes principales de programación.

Módulo 2: Creando un punto de venta en internet


En este punto comenzaremos a tomar conceptos de JavaScript y conoceremos MySQL que es un maravilloso sistema de gestión de bases de datos relacional, vamos a crear una tienda virtual el móduo ideal para los emprendedores. Conoceremos conceptos como node.js, express, y aprenderemos a publicar nuestros trabajos en repositorios de Git Hub. Competencia del taller: Programar y diseñar un punto de venta en internet usando JavaScript que pueda ser visto en la web.
